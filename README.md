# Black Jack CLI 

## Technologies used 
- Python

## Project Description 
Little black jack game you can play in your terminal. 
I made it as a little fun break from dealing with React. 

## Features
- Betting system
- Dealer AI
- Randomized card deck
- Hand value calculation
- Win/Loss detection

## Installation
- Clone the repository:
    - git clone <https://github.com/BaderMajidAlRefai/blackjack>
- Open the project folder in your terminal:
    - cd blackjack
    - Run the game:
        - python3 main.py

## Requirements 
- Python3
